## 命題論理の定義いろいろ
### 公式な記号文の定義
記号文は以下に示す語彙を並べたものであってかつ以下に示すルールに従って構成されるものである。
語彙
- 文記号:`P`,`Q`,...,`Z`
  - 但し文記号が不足した場合`P_1`など下付き添字を許容する
- 結合子:`~`,`∧`,`∨`,`→`,`↔`
- カッコ:`(`,`)`
ルール
1. 文記号はそれ単独で記号文である。(文記号単独からなる記号文を原子文という)
2. `φ`が記号文であるとき`~φ`は記号文である
3. `φ`と`ψ`が記号文であるとき、`(φ∧ψ)`,`(φ∨ψ)`,`(φ→ψ)`,`(φ↔ψ)`はそれぞれ記号文である
4. 記号文中の空白,タブ文字などについては全て無視する

### 非公式な記号文の定義
公式な記号文の表記を簡略化することを目的として以下のように非公式な記号文を定める
1. 記号文全体にかかる括弧は省いて良い
2. 括弧`(`,`)`の代わりに括弧`[`,`]`を用いて良い
3. 結合子`∧`,`∨`は結合子`→`,`↔`よりも左右の結合が強いと取り決め、括弧を省いて良い
4. 左寄せの規則(後述)を約束し、括弧を省いて良い
   - 左寄せの規則とは
     - k個の記号文`φ_1,φ_2,...,φ_k`のすべてを結合子`∧`でつなげた記号文`φ_1∧φ_2∧,...,∧φ_k`においては、`∧`が左から順に適用されると約束する。
     - k個の記号文`φ_1,φ_2,...,φ_k`のすべてを結合子`∨`でつなげた記号文`φ_1∨φ_2∨,...,∨φ_k`においては、`∨`が左から順に適用されると約束する。
     - 但し、結合子`∧`と結合子`∨`が同じ優先順位で同時に出てくる場合(ex. `φ_1∧φ_2∧φ_3∨φ_4`)においてはこれの適用外であるため、括弧を省いてはならない。
上記以外の非公式化は認めない。
非公式な記号文の条件にすら含まれない記号列は記号文ではない。

### 真偽値表
記号文に出てくる文記号の真偽の組全てにおいて推論の仮定と結論それぞれの真偽を記したもの
ある推論の仮定が`k`個で記号文全体を通して`n`個の文記号が出現していたならば、真偽値表は`2^n`行(文記号全ての真偽を確認するため)`n+k+1`列(文記号,仮定,結論と並べるため)の行列として表される

### 意味論的妥当性の判断
上記で定義された真偽値表の行の中に、全ての仮定が`True`であるにも関わらず結論が`False`になるような行が無ければ、その推論は意味論的に妥当と言える。
