# 科学哲学B (論理学入門)

## 要件
- [ ] 任意の命題論理の記号文を意味論的に妥当かどうか判断する
  - [ ] 任意の記号列が記号文かどうか判断する
  - [ ] 任意の記号文が公式な記号文か判断する
  - [ ] 非公式であれば公式にする
  - [ ] 前提で使われるすべての文記号についてそれぞれ真偽表を作成する
  - [ ] 記号文の結合子の順に真偽値を決定する
  - [ ] 前提がすべて真だが結論が偽である行の有無を確認する
- [ ] 意味論的に妥当な命題論理の記号文についてその推論を統語論的に示す
  - [ ] 公式な記号文を用いてテキストのヒントの順に演繹を構成する
    - ただし使用する推論規則は{MP,MT,DN,R,Adj,S,Add,MTP,BC,CB,DM,CDJ}とする
    - ただし使用する演繹の形式は{CD,ID}とする
  - [ ] (optional) できるだけ行数の少ない統語論的な演繹を目指す
- [ ] 任意の述語論理の記号文を意味論的に妥当かどうか判断する
  - [ ] 任意の記号列が記号文かどうか判断する
  - [ ] 任意の記号文が公式な記号文か判断する
  - [ ] 非公式であれば公式にする
  - [ ] 有限の反証モデルの作成を試みる
    - ただし理論上不可能なのでドメインが4つぐらいまでで諦める
  - [ ] 前提で使われるすべての文記号についてそれぞれ真偽表を作成する
  - [ ] 記号文の結合子の順に真偽値を決定する
  - [ ] 前提がすべて真だが結論が偽である行の有無を確認する
  - [ ] 無限の反証モデルの作成を試みる
    - 運ゲーなのでLLMの力を借りてみる
- [ ] 意味論的に妥当な述語論理の記号文についてその推論を統語論的に示す
  - [ ] 公式な記号文を用いてテキストのヒントの順に演繹を構成する
    - ただし使用する推論規則は{MP,MT,DN,R,Adj,S,Add,MTP,BC,CB,DM,CDJ,UI,EI,EG,QN}とする
    - ただし使用する演繹の形式は{CD,ID,UD}とする
